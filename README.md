# Offside AI

Upload a football clip, get an AI offside verdict with a shareable 3D animation.

## Deploy on Railway

1. Upload this ZIP to GitHub (new repo)
2. Connect the repo to Railway
3. Railway auto-detects Node.js + FFmpeg via nixpacks.toml
4. No environment variables needed — it works out of the box

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:5000
